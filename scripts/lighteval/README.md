# LightEval

This directory should contain the LightEval library (customized version for LFM-MoE models).

## Setup

Clone the official lighteval repo at the specific tag used:

```bash
git clone --branch v0.10.0 --depth 1 https://github.com/huggingface/lighteval.git scripts/lighteval
```

Then install locally:

```bash
pip install scripts/lighteval
```
